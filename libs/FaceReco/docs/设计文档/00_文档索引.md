## 📡 ZMQ 话题速查

### 命令 (Daemon → Vision Service)

| 话题 | 描述 |
|------|------|
| `cmd.vision.enable` | 启用/禁用服务 |
| `cmd.vision.mode` | 设置运行模式 |
| `cmd.vision.face.register` | 注册人脸 |
| `cmd.vision.face.update` | 更新人脸 |
| `cmd.vision.face.delete` | 删除人脸 |
| `cmd.vision.capture.photo` | 拍照 |

### 事件 (Vision Service → Daemon)

| 话题 | 描述 |
|------|------|
| `event.vision.face` | 人脸快照 |
| `event.vision.face.recognized` | 人脸识别成功 |
| `event.vision.face.new` | 新人脸出现 |
| `event.vision.face.lost` | 人脸消失 |

## 🚀 快速开始

### 1. 运行单元测试

```bash
cd /home/pi/dev/dolydev
python3 scripts/test_face_reco_manager.py
```

### 2. 运行集成测试

```bash
# 确保没有其他进程使用 doly_events.sock
python3 scripts/test_face_reco_integration.py
```

### 3. 启动 Vision Service

```bash
cd /home/pi/dolydev/libslibs/FaceReco/build
./LiveFaceReco
```

### 4. E2E/性能/稳定性脚本

```bash
python3 scripts/test_vision_e2e.py
python3 scripts/test_vision_performance.py
python3 scripts/test_vision_stability.py
```
