# 安装

Doly 在机器人镜像中**预装**了 Python SDK（适用于 Raspberry Pi OS，Python 3.11）。

## 在机器人上使用 SDK

大多数用户可以直接开始：

- 导入模块（例如 `import doly_arm`）
- 从仓库运行模块的 `example.py` 作为起点
- 基于示例编写自己的脚本

## 开发者安装（从源码重建）

如果修改了模块源码，需要在机器人上重新构建并安装该模块。

典型工作流程（每个模块）：

1. 编辑 `source/bindings.cpp`
2. 使用模块的 `pyproject.toml` 构建 wheel
3. 在机器人上使用 `pip` 安装该 wheel

> 具体构建命令可能会因工具链和 C++ 库位置不同而异。我们在每个模块的仓库中记录了推荐的构建步骤，本页专注于用户级别的安装说明。
