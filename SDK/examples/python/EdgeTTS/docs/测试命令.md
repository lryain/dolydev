## 运维脚本

为便捷管理，我们在 `libs/tts/edge-tts` 下新增 `scripts/manage_service.sh`，参考了 `libs/drive/scripts/manage_service.sh` 的结构。该脚本会：

- 检查 `python3` 与 `zmq_tts_service.py`；
- 通过 `python3 -m pip install -r requirements.txt` 安装依赖；
- 安装/复制 `edge-tts.service` 到 `/etc/systemd/system`，并启用 `edge-tts-service`；
- 提供 `prepare|install|uninstall|start|stop|restart|status|logs|monitor|reload|redeploy` 进维操作。

使用示例：
```bash
cd libs/tts/edge-tts
./scripts/manage_service.sh install
./scripts/manage_service.sh start
./scripts/manage_service.sh logs
```
