## 快速执行

### 1. 编译

```bash
cd /home/pi/dolydev/libs/FaceReco/build
make -j2
```

### 2. 运行可执行功能测试

```bash
cd /home/pi/dolydev/libs/FaceReco
scripts/run_vision_zmq_functional_test.sh
```

### 3. 查看结果

```bash
tail -n 80 /home/pi/dolydev/libs/FaceReco/tmp/functional_zmq_test/functional_test.log
```

最近一次完整执行结果为: all functional tests passed。
