> **版本**: 1.0  
> **日期**: 2026-02-08  

---

## 1. 通信架构

### 1.1 Socket 类型

| Socket | 类型 | 端点 | 用途 |
|--------|------|------|------|
| Publisher | PUB | `ipc:///tmp/doly_event_bus.sock` | 发布全局事件和状态 |
| Subscriber | SUB | `ipc:///tmp/doly_event_bus.sock` | 订阅全局事件和命令 |
| Reply | REP | `ipc:///tmp/doly_vision_query.sock` | 响应Vision服务查询请求 |

### 1.2 消息信封格式

所有消息遵循统一的 JSON 信封格式：

```json
{
  "ts": 1738972800000,           // Unix 毫秒时间戳
  "src": "vision_service",       // 消息来源
  "request_id": "uuid-xxxx",     // 请求 ID（可选，用于请求-响应匹配）
  "version": "1.0",              // 协议版本
  "data": {                      // 消息负载
    // 具体内容
  }
}
```

## 2. 命令协议 (Commands)

### 2.1 服务控制

#### cmd.vision.enable
启用或禁用视觉服务。

```json
// Request
{
  "enabled": true,       // true=启用, false=禁用
  "reason": "user_request"  // 可选：禁用原因
}

// Response (via event.vision.module.state)
{
  "state": "running",    // running, paused, disabled
  "enabled": true
}
```

#### cmd.vision.mode
设置运行模式。

```json
// Request
{
  "mode": "FULL",        // IDLE, STREAM_ONLY, DETECT_ONLY, DETECT_TRACK, FULL, CUSTOM
  "modules": [           // 仅 CUSTOM 模式需要
    "CAMERA_CAPTURE",
    "FACE_DETECTION",
    "VIDEO_STREAM"
  ]
}

// Response (via event.vision.module.state)
{
  "mode": "FULL",
  "active_modules": ["CAMERA_CAPTURE", "FACE_DETECTION", "FACE_RECOGNITION", "VIDEO_STREAM"],
  "state": "running"
}
```

#### cmd.vision.module.load
加载指定模块。

```json
// Request
{
  "module": "FACE_RECOGNITION",  // 模块名称
  "config": {                    // 可选：模块配置
    "threshold": 0.85
  }
}

// Response (via event.vision.module.state)
{
  "module": "FACE_RECOGNITION",
  "state": "running",
  "success": true
}
```

#### cmd.vision.module.unload
卸载指定模块。

```json
// Request
{
  "module": "FACE_RECOGNITION"
}

// Response (via event.vision.module.state)
{
  "module": "FACE_RECOGNITION",
  "state": "unloaded",
  "success": true
}
```

### 2.2 配置控制

#### cmd.vision.config

```json
// Request

  "params": {
    "face_thre": 0.45,

  "updated_params": ["face_thre", "angle_threshold", "enable_video_stream_to_eyeengine"]
}
```

### 2.3 人脸管理

#### cmd.vision.face.register
注册新人脸。

```json
// Request - 方式1：使用当前检测到的人脸
{
  "method": "current",
  "tracker_id": 5,       // 当前跟踪的人脸 ID
  "name": "张三",
  "metadata": {
    "relation": "friend",
    "notes": "朋友"
  }
}

// Request - 方式2：使用图片文件
{
  "method": "file",
  "file_path": "/home/pi/photos/zhangsan.jpg",
  "name": "张三",
  "metadata": {
    "relation": "friend"
  }
}

// Response (via event.vision.face.registered)
{
  "success": true,
  "face_id": "face_001",
  "name": "张三",
  "message": "人脸注册成功"
}
```

#### cmd.vision.face.update
更新人脸信息。

```json
// Request
{
  "face_id": "face_001",     // 或使用 name 查找
  "name": "zhangsan",        // 用于查找
  "updates": {
    "name": "张三三",         // 新名字
    "metadata": {
      "relation": "family",
      "notes": "更新备注"
    }
  }
}

// Response (via event.vision.face.updated)
{
  "success": true,
  "face_id": "face_001",
  "updated_fields": ["name", "metadata"]
}
```

#### cmd.vision.face.delete
删除人脸数据。

```json
// Request
{
  "face_id": "face_001",     // 或
  "name": "张三",            // 按名字删除
  "delete_images": true      // 是否同时删除图片文件
}

// Response (via event.vision.face.deleted)
{
  "success": true,
  "face_id": "face_001",
  "name": "张三",
  "deleted_files": 3
}
```

#### cmd.vision.face.recapture
重新采集人脸。

```json
// Request
{
  "face_id": "face_001",
  "tracker_id": 5,           // 当前跟踪的人脸 ID
  "append": true             // true=追加, false=替换
}

// Response (via event.vision.face.updated)
{
  "success": true,
  "face_id": "face_001",
  "total_samples": 4
}
```

### 2.4 拍照录像

#### cmd.vision.capture.photo
拍照。

```json
// Request
{
  "request_id": "photo_001",
  "save_path": "/home/pi/photos/",
  "filename": "my_photo.jpg",    // 可选
  "include_annotations": true,   // 包含人脸框标注

  "format": "jpg"                // jpg, png
}

// Response (via event.vision.capture.complete)
{
  "request_id": "photo_001",
  "type": "photo",
  "success": true,
  "file_path": "/home/pi/photos/my_photo.jpg",
  "file_size": 125432,
  "resolution": {"width": 320, "height": 240},
  "faces_detected": 1
}
```

#### cmd.vision.capture.video.start
开始录像。

```json
// Request
{
  "request_id": "video_001",
  "save_path": "/home/pi/videos/",
  "filename": "my_video.mp4",
  "max_duration_seconds": 60,    // 最大时长
  "include_annotations": false,
  "codec": "h264",               // h264, mjpeg
  "fps": 15
}

// Response (via event.vision.capture.started)
{
  "request_id": "video_001",
  "type": "video",
  "status": "recording",
  "start_time_ms": 1738972800000
}
```

#### cmd.vision.capture.video.stop
停止录像。

```json
// Request
{
  "request_id": "video_001"
}

// Response (via event.vision.capture.complete)
{
  "request_id": "video_001",
  "type": "video",
  "success": true,
  "file_path": "/home/pi/videos/my_video.mp4",
  "duration_seconds": 15.5,
  "file_size": 2345678
}
```

### 2.5 视频流控制

#### cmd.vision.stream
控制视频流输出。

```json
// Request

  "action": "start",         // start, stop, pause, resume
```

## 3. 事件协议 (Events)

### 3.1 人脸事件

#### event.vision.face
人脸检测快照（周期性发布）。

```json
{
  "event": "face_snapshot",
  "count": 2,
  "frame_size": {"width": 320, "height": 240},
  "primary": {
    "id": 5,
    "bbox": {"x1": 100, "y1": 80, "x2": 200, "y2": 180},
    "normalized": {"x": 0.47, "y": 0.54},
    "confidence": 0.92,
    "name": "张三",
    "liveness": true,
    "angle": 8.5
  },
  "faces": [
    {
      "id": 5,
      "bbox": {"x1": 100, "y1": 80, "x2": 200, "y2": 180},
      "normalized": {"x": 0.47, "y": 0.54}
    },
    {
      "id": 6,
      "bbox": {"x1": 220, "y1": 90, "x2": 300, "y2": 170},
      "normalized": {"x": 0.81, "y": 0.54}
    }
  ]
}
```

#### event.vision.face.recognized
人脸识别成功事件。

```json
{
  "event": "face_recognized",
  "tracker_id": 5,
  "face_id": "face_001",
  "name": "张三",
  "confidence": 0.95,
  "liveness": true,
  "bbox": {"x1": 100, "y1": 80, "x2": 200, "y2": 180},
  "normalized": {"x": 0.47, "y": 0.54},
  "metadata": {
    "relation": "friend"
  },
  "timestamp_ms": 1738972800000
}
```

#### event.vision.face.new
新人脸出现事件（未识别的新人脸稳定出现）。

```json
{
  "event": "face_new",
  "tracker_id": 7,
  "detection_count": 5,       // 连续检测次数
  "bbox": {"x1": 150, "y1": 100, "x2": 250, "y2": 200},
  "liveness": true,
  "timestamp_ms": 1738972800000
}
```

#### event.vision.face.lost
人脸消失事件。

```json
{
  "event": "face_lost",
  "tracker_id": 5,
  "name": "张三",             // 如果已识别
  "last_seen_ms": 1738972798000,
  "lost_duration_ms": 2000,
  "timestamp_ms": 1738972800000
}
```

## 4. 查询协议 (Queries)

使用 REQ/REP 模式的同步查询。

### query.vision.face.list
查询人脸数据库。

```json
// Request
{
  "filter": {
    "relation": "friend"     // 可选过滤条件
  },
  "limit": 100,
  "offset": 0
}

// Response
{
  "success": true,
  "total": 25,
  "faces": [
    {
      "face_id": "face_001",
      "name": "张三",
      "created_at": "2026-02-01T10:00:00Z",
      "last_seen": "2026-02-08T09:30:00Z",
      "metadata": {"relation": "friend"},
      "sample_count": 3
    },
    // ...
  ]
}
```

### query.vision.face.position
查询当前人脸位置。

```json
// Request
{
  "tracker_id": 5,           // 可选，指定跟踪 ID
  "name": "张三"             // 可选，按名字查询
}

// Response
{
  "success": true,
  "faces": [
    {
      "tracker_id": 5,
      "name": "张三",
      "bbox": {"x1": 100, "y1": 80, "x2": 200, "y2": 180},
      "normalized": {"x": 0.47, "y": 0.54},
      "last_update_ms": 1738972800000
    }
  ]
}
```

## 5. 错误处理

### 5.1 错误响应格式

```json
{
  "success": false,
  "error": {
    "code": "FACE_NOT_FOUND",
    "message": "未找到指定的人脸记录",
    "details": {
      "face_id": "face_999"
    }
  }
}
```

### 5.2 错误码表

| 错误码 | 描述 |
|--------|------|
| `OK` | 成功 |
| `INVALID_PARAM` | 参数无效 |
| `MODULE_NOT_LOADED` | 模块未加载 |
| `MODULE_LOAD_FAILED` | 模块加载失败 |
| `FACE_NOT_FOUND` | 人脸不存在 |
| `FACE_ALREADY_EXISTS` | 人脸已存在 |
| `NO_FACE_DETECTED` | 未检测到人脸 |
| `CAMERA_ERROR` | 摄像头错误 |
| `STORAGE_ERROR` | 存储错误 |
| `TIMEOUT` | 操作超时 |
| `SERVICE_DISABLED` | 服务已禁用 |

## 6. 订阅前缀

Vision Service 订阅以下前缀的命令：

```cpp
const char* prefixes[] = {
    "cmd.vision",      // 所有视觉服务命令
    "cmd.eye.follow"   // 眼神跟随控制
};
```

Daemon 中的 FaceRecoManager 订阅以下前缀：

```python
prefixes = [
    "event.vision.face",       # 人脸事件
    "status.vision.state"      # 状态快照
]
```
